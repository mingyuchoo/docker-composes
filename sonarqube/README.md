# SonarQube

## 준비할 환경

- ubuntu 18.04 이상인 운영체제
- node v12 버전 이상 설치
- docker 설치
- docker-compose 설치


## How to start this container

```bash
docker compose up -d
# or
npm run docker:up
```

## How to stop and delete container

```bash
docker compose down
# or
npm run docker:down
```

## 접속 방법

- `http://localhost:9000`
- id/pw: admin/admin
