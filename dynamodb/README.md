# Docker Compose for DynamoDB

## How to start this container

```bash
docker composeup -d
# or
npm run docker:up
```

## How to stop and delete container

```bash
docker composedown
# or
npm run docker:down
```

## How to use DynamoDB Admin GUI

```bash
npm install
npm run admin
```

connect to <http://localhost:8001>
